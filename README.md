# GLiNER Medical – Données Synthétiques & Extraction d’Entités

##  Objectif
Créer un corpus médical synthétique annoté pour l’extraction d’entités médicales
avec GLiNER (sans données sensibles réelles).

##  Structure
- data/raw : données JSON annotées
- data/stats : statistiques par entité
- scripts : génération, analyse, extraction, fine-tuning
- docs : documentation méthodologique

##  État du projet
- Génération des données
- Analyse des entités
- Extraction avec GLiNER pré-entraîné
- Fine-tuning : en attente d’équilibrage des entités
